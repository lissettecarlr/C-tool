# software_tool

## 1、介绍

这是一个集成多种小工具的软件包，例如crc校验，产生随机数，字节变换等C语言工具，方便底层嵌入式代码的编写。

### 1.1 目录结构


| 名称 | 说明 |
| ---- | ---- |
| docs  | 文档目录 |
| examples | 例子目录，并有相应的一些说明 |
| include  | 头文件目录 |
| src  | 源代码目录 |


## 2、如何打开 software_tool

使用 software_tool package 需要在 RT-Thread 的包管理器中选择它.

然后让 RT-Thread 的包管理器自动更新，或者使用 `pkgs --update` 命令更新包到 BSP 中。
